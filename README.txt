1. Install Python and pip
2. Navigate to the source code directory of our application
3. Run the following pip commands:
    pip install pymongo
    pip install flask
    pip install gevent
    pip install Flask-SocketIO
4. Run “python backend.py”
5. Open a web browser and go to “localhost:5000”
6. Log in using whatever credentials you would like
    Cashier:
        USER: “Adam”
        PASS: “a”
    Barista:

        PASS: “c”
    Manager:
        USER: “Zack”
        PASS: “z”
7. Try it out!
